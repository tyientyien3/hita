# HITA

This application is for those who plan to go and stay south east asia.
<br>
You can ask anything that you are wondering about east asia real life.

# Requirement
* Ruby :2.5.7
* Rails:5.2.4.1

# Usage

```bash
git clone https://github.com/tyientyien3/hita.git
cd hita
bundle install
rails db:migrate
rails db:seed
```
